# 贡献指南

如果你愿意一起完善这个项目，这里是我整理的贡献说明。

## 开发环境

- Java 17
- Maven 3.9+
- MySQL 8.0+

## 本地启动

1. 复制 `.env.example` 为 `.env`，填入数据库和模型配置。
2. 启动依赖：`docker compose up -d mysql`
3. 启动应用：`mvn spring-boot:run`

## 提交建议

- 保持 UTF-8 编码。
- 优先提交最小、可验证的改动。
- 新增功能时补充测试或复现说明。
- 涉及索引变更能力时，默认按只读演示模式处理，先不要直接打开真实变更。

## Pull Request

- 说明问题背景和目标用户场景。
- 写清验证方式。
- 如果涉及 UI，尽量附截图或录屏。
